API Documentation

Learn how to use our authentication API

Introduction

This API provides authentication functionalities including signup, login, logout, password reset, and more.

Endpoints

Signup

URL: /api/auth/signup

Method: POST

Description: Create a new user account.

{
    "fullname": "string",
    "email": "string",
    "password": "string",
    "role": "string" (optional, default: "user")
}

Login

URL: /api/auth/login

Method: POST

Description: Authenticate a user and retrieve a token.

{
    "email": "string",
    "password": "string"
}

Logout

URL: /api/auth/logout

Method: POST

Description: Logout the user.

Forgot Password

URL: /api/auth/forgot-password

Method: POST

Description: Send a password reset link to the user's email.

{
    "email": "string"
}

Reset Password

URL: /api/auth/reset-password/:token

Method: POST

Description: Validate a password reset token.

{
    "token": "string"
}

Set New Password

URL: /api/auth/set-new-password

Method: POST

Description: Set a new password for the user.

{
    "userId": "string",
    "newPassword": "string"
}
